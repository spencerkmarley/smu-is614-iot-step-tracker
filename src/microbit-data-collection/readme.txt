
Please choose either 1 of the below to run: 

receive_csv.py : reads the serial input from receiver microbit and saves as csv locally 
(please read instructions in comments)

receive_csv_to_s3.py : reads the serial input from receiver microbit and saves as csv locally + auto upload file to s3
(please read instructions in comments)

_______________________________________________________________________________________

Both of these scripts need to be loaded to microbit: 
receiver.py : copy this to the make code console of the receiver microbit 
transmitter.py : copy this to the make code of the transmitter microbit 

NOTE: Transmitter micorbit needs to be V1 >> since V2 is not compatible
